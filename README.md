# 文字列探索アルゴリズム 可視化ツール

React 18とTypeScript、Atomic Designパターンで構築された文字列探索アルゴリズムの可視化アプリケーションです。

## デモプレイ
https://yunbow.github.io/react-sim-string-search/demo/

## 主要機能

### 1. 文字列探索機能

- **対応アルゴリズム**
  - 力まかせ法（Brute-Force）
  - クヌース–モリス–プラット法（KMP）
  - ボイヤー・ムーア法（Boyer–Moore / BM）

- **可視化要件**
  - 探索処理のステップを逐次可視化
  - 比較・一致・不一致などの状態を毎ステップ表示
  - 各アルゴリズムの特徴を色分け・ハイライトで可視化

### 2. データ生成機能
- **データ生成パターン**
  - ランダム：ランダムに値を列挙
  - 反復パターン：パターンが繰り返し出現する文字列
  - 部分一致多数：部分一致が多く含まれる文字列
  - 最悪ケース：アルゴリズムの最悪ケースとなる文字列

- **データサイズ**
  - テキスト長：1〜100文字（デフォルト：20文字）
  - パターン長：1〜50文字（デフォルト：3文字）

### 3. 探索実行機能
- **実行制御**
  - ボタンラベル遷移：「実行」→「一時停止」→「再開」
  - リセットボタン：データ生成状態を初期化

- **探索速度調整**
  - アニメーション速度を調整できるスライダー（0.25x〜3.0x）

### 4. アニメーション表示機能
- **文字表示**
  - テキストとパターンを文字単位で表示

- **色分けルール**
  | 状態 | 色 | 説明 |
  |------|----|------|
  | 未処理 | 灰色 | まだ処理されていない文字 |
  | 比較中 | 青色 | 現在比較中の文字 |
  | 一致 | 緑色 | 一致した文字 |
  | 不一致 | 赤色 | 不一致の文字 |
  | 発見 | 黄色 | パターンが見つかった位置 |
  | パターン位置 | 橙色 | 現在のパターン位置（Boyer-Moore） |

### 5. UI操作機能
- アルゴリズム選択プルダウン
- データパターン選択プルダウン
- データサイズ入力欄
- 実行／一時停止／再開／リセットボタン
- アニメーション速度スライダー
- **アルゴリズム比較モード**
  - 左右にアルゴリズム選択プルダウンを2つ配置
  - 初期データは完全に同一データセットを使用
  - 2つの探索を同時に実行可能

### 6. ログ・状態表示機能
- **探索状態表示**
  - ステップ数
  - 比較回数
  - シフト回数
  - 実行時間（ミリ秒）
  - 一致位置

- **実行ログ**
  - 各ステップの詳細情報を表示
  - 最新のログを強調表示

### 7. 計算量表示機能
- 各アルゴリズムの理論的計算量（最悪・平均・最良）をUIに表示
  - 例：力まかせ法 → Worst: O(n×m), Average: O(n×m), Best: O(n)

## 技術スタック

- **React 18** - UIライブラリ
- **TypeScript** - プログラミング言語
- **Storybook 7** - コンポーネント開発・ドキュメント
- **CSS Modules** - スタイリング
- **Vite** - ビルドツール

## プロジェクト構造

```
src/
├── features/                        # 機能別モジュール
│   └── search/                      # 文字列探索機能
│       ├── algorithms/              # 探索アルゴリズム実装
│       │   ├── bruteForce.ts
│       │   ├── kmp.ts
│       │   ├── boyerMoore.ts
│       │   └── index.ts
│       ├── components/              # 機能専用コンポーネント
│       │   ├── StringVisualizer/   # 文字列可視化
│       │   ├── SearchControls/     # 探索コントロール
│       │   └── SearchStats/        # 統計表示
│       ├── hooks/                   # カスタムフック
│       │   └── useSearchSimulator.ts
│       ├── utils/                   # ユーティリティ
│       │   └── dataGenerator.ts
│       ├── SearchSimulator/         # シングルモードコンポーネント
│       └── ComparisonMode/          # 比較モードコンポーネント
├── components/                      # 共通UIコンポーネント
│   ├── Button/                      # ボタン
│   ├── Select/                      # セレクトボックス
│   ├── Slider/                      # スライダー
│   └── Input/                       # テキスト入力
├── types/                           # 汎用的な型定義
│   └── index.ts
├── App.tsx                          # メインアプリ
├── main.tsx                         # エントリーポイント
└── theme.css                        # テーマカラー定義
```

## スクリプト

```bash
# セットアップ
npm install

# 開発サーバー起動
npm run dev

# ビルド
npm run build

# プレビュー
npm run preview

# Storybook起動
npm run storybook

# Storybook ビルド
npm run build-storybook
```
